# 📘 Part 05. 배열, 포인터, 참조

1. 개념 설명 (Concept)
배열 (Array)
같은 자료형의 데이터를 연속적으로 저장하는 자료구조. 인덱스를 통해 접근 (0부터 시작). 예: int arr[3] = {10, 20, 30};

포인터 (Pointer)
메모리 주소를 저장하는 변수. * : 포인터 선언 및 역참조 연산자. & : 변수의 메모리 주소 연산자. 예: int* p = &a; → 변수 a의 주소 저장, *p → 포인터가 가리키는 값.

참조 (Reference)
변수의 또 다른 이름(별칭)을 부여. & 기호로 선언 (포인터와 다름). 예: int& ref = a; → ref는 a의 별칭.

2. 코드 예제 (Code Example)
배열 예제, 포인터 예제, 참조 예제가 각각 제공되며, 실행 결과와 함께 배열 합 구하기, 포인터를 이용한 배열 출력 미션이 포함된다.

5. 단계별 해설 (Step by Step)
배열: 같은 자료형 데이터를 연속적으로 저장하고, 인덱스로 접근. 포인터: 변수의 메모리 주소를 저장, *p로 접근 시 값 참조. 참조: 변수의 또 다른 이름을 부여, 원본과 동일하게 작동.

7. 요약 (Summary)
배열: 같은 자료형을 연속적으로 저장, 인덱스로 접근. 포인터: 변수의 메모리 주소를 저장하고, * 연산자로 값에 접근. 참조: 변수의 또 다른 이름(별칭), 원본과 동일하게 동작. 배열, 포인터, 참조는 메모리 개념 이해와 고급 프로그래밍의 기초.
