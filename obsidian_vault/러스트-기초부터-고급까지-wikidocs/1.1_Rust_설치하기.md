# 1.1 Rust 설치하기

환경 구축은 Windows 11 운영체제를 가정하고 설명한다.
Microsoft C++ Build Tools 설치
Rust를 컴파일하려면 Microsoft C++ Build Tools가 필요하다. 이것을 먼저 설치하고 Rust를 설치하는게 편하다.
1) 아래 링크에 가서 Build Tools 버튼을 눌러 다운로드 한다.
https://visualstudio.microsoft.com/ko/visual-cpp-build-tools/
2) 다운로드 파일 실행
다운로드한 파일을 실행 후, 아래와 같이 C++를 사용한 데스크톱 개발에 체크하고 선택 사항들이 체크되어 있는지 확인한다.
3) 설치 한다.
Rust 다운로드 및 설치
1) 아래 사이트에서 Rust 설치 프로그램을 다운로드
자신의 윈도우즈 운영체제 비트 수에 맞게 32비트 혹은 64비트 설치 프로그램을 다운로드 해야한다.
다운로드 사이트: https://www.rust-lang.org/tools/install
2) 설치
위 1번에서 다운로드한 rustup-init.exe파일을 실행해서 설치한다. 모든 옵셥을 디폴트로 선택하고 설치.
3) 설치 확인
명령 프롬프트(cmd 창) 혹은 PowerShell을 열고 rustc --version 타이핑 후 엔터
rustup update
rust 컴파일러를 최신 버전으로 업데이트하는 명령어.
Rust를 설치하고 한 번 해주는 게 좋다고 한다. 근데, 안해도 큰 문제 없다.
rustup self uninstall
Rust를 제거하는 명령이다. 관련 파일들과 환경 변수들이 삭제된다.
rustup default 1.81
특정 버전의 Rust를 설치하는 명령어다.
