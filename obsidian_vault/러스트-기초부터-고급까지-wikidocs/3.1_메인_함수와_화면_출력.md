# 3.1 메인 함수와 화면 출력

메인 함수 : fn main()
C 언어에서와 같이 Rust에서도 프로그램의 시작점은 main 함수 부터이다. Rust로 프로그램 코드를 짜서 exe 파일로 만들면 main 함수 부터 실행된다는 의미다.

main 함수의 형태
```rust
fn main() {
  println!("hello");
}
```
리턴 타입이 없다. 파라미터로 받는 것이 없다. 아주 간단한 형태이다. fn main()으로 족하다.

main 함수의 리턴
일반적으로 main 함수는 리턴 타입 없이 짜도 무방하다. 그러나, 만약 실행 결과를 리턴하고 싶으면 Result 타입으로 리턴해야 한다.
```rust
fn main() -> Result<(), i32> {
  ...
  Ok(())
}
```

main 함수에서 파라미터 받기
Rust의 경우는 main 함수의 파라미터로 받지 않는다. 다른 방식을 사용한다. env::args().collect()를 사용해서 입력값을 받는다.
```rust
use std::env;

fn main() {
    let args: Vec<String> = env::args().collect();
    let src_file = &args[1];
    let tgt_file = &args[2];
    println!("Source File: {}", src_file);
    println!("Target File: {}", tgt_file);
}
```
&args[0]에는 프로그램 이름이 들어간다.

화면 출력 : print!, println!
Rust에서는 print!과 println!이 매크로다. 메시지를 출력하고 나서 줄을 바꾸는 것은 println!, 줄 바꿈 없는 것은 print!다.
```rust
fn main() {
    println!("hello");
    let sum = 100;
    println!("a+b={}", sum);
    println!("a+b={sum}");
    let v = vec![1,2,3];
    println!("{:?}", v);
}
```
