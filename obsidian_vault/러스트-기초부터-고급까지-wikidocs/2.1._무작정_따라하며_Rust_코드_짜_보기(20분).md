# 2.1. 무작정 따라하며 Rust 코드 짜 보기(20분)

1) 아직 Rust 코딩을 위한 개발 환경 구성이 안 되어 있으면 프로그래밍 환경 구축하기에 가서 Rust와 VS Code를 설치한다.
2) cmd 창을 열어서, 앞으로 문제 풀이 폴더로 사용할 "c:/work_rust/euler" 폴더로 이동한다. 폴더가 안 만들어져 있으면 만든다.
3) cmd 창의 "c:/work_rust/euler" 에서 cargo new ex1 이라고 타이핑하고 엔터. "Created binary..."라는 메시지가 안 나오면 Rust 설치가 잘못된 것이다.
4) 위 3번 명령어에 의해서 ex1 폴더가 생성되었다.
5) "ex1" 폴더에 가서 code . 타이핑하고 엔터. VS Code를 실행시킨다.
6) VS Code가 실행되면서 "Trust The authors of ..." 메시지가 나오면 해당 항목을 체크하고 "Yes, I trust the authors" 버튼을 클릭한다.
7) VS Code에서 왼편 EXPLORER 창에서 src 폴더에 있는 main.rs를 클릭한다.
8) fn main() 위에 있는 "Run" 버튼을 클릭하여 Hello, world!가 출력되는지 확인한다.
9) "main.rs" 파일의 내용을 아래 코드로 교체한다.

fn main() {    
    println!("1+...+100={}",get_sum(100));
}

fn get_sum(n:u32) -> u32 {
    let mut sum:u32 = 0;

    for i in 1..=n {
        sum += i;
    }

    return sum;
}

10) fn main() 위에 있는 "Run"을 눌러서 1부터 100까지 합이 출력되는지 확인한다.
