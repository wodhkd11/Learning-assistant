# 2.2 작성한 코드 이해하기 (20분)

앞 페이지에서 타이핑한 코드는 아래와 같다. 1~100까지의 수를 더한 값을 출력하는 코드이다.

```rust
fn main() {
    println!("1+...+100={}",get_sum(100));
}

fn get_sum(n:u32) -> u32 {
    let mut sum:u32 = 0;

    for i in 1..=n {
        sum += i;
    }

    return sum;
}
```

1. main 함수
main 함수는 해당 코드가 컴파일되어 실행파일이 될 때 가장 먼저 실행된다. 모든 실행 프로그램은 main 함수를 가진다. "cargo new"로 생성된 main.rs에는 기본 Hello World 코드가 들어 있다.

2. 화면 출력
Rust에서는 println! 매크로를 사용한다. println!은 함수가 아니라 매크로이며, 느낌표(!)로 끝난다. 변수 값 출력은 {} 또는 {변수명} 형태로 가능하다.

3. 함수의 호출
함수 안에서 다른 함수를 호출할 수 있다. get_sum(100)을 호출하면 5050이 반환된다.

4. 함수
fn 키워드로 선언하며, 함수명(소문자+언더스코어 권장), 파라미터(타입 명시 필수), -> 리턴 타입으로 구성된다. return 키워드 없이 마지막 표현식만 적어도 리턴된다.

5. for 루프
for i in 1..=n 형태로 사용하며, 1..=100과 1..101은 동일하다. 중괄호는 필수이다.
