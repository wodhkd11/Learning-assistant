변수
모든 프로그래밍 언어는 어떤 데이터를 담는 변수(variable)를 사용할 수 있게 한다. 실제로는 어떤 메모리 공간을 가리킨다. Rust에서는 let 키워드를 사용해서 변수를 생성할 수 있다. 다른 언어와 다른 가장 큰 특징은 생성된 변수가 '불가변성'이라는 거다. 즉, 읽기 전용이다. 해당 변수에 대해 '쓰기(write)'도 가능하게 하려면 mut 키워드를 추가해서 변수명 앞에 let mut이라고 해야한다.

Rust에서 변수의 선언
Rust에서 변수 선언은 let 키워드를 사용한다. 일반적인 방식은 let <변수명>:<타입> = <값> 혹은 let <변수명>:<타입>과 같이 값 지정 없이 선언할 수도 있다.

let 변수명 : 타입 = 값;
let 변수명 : 타입;   // 초깃값 지정없이 생성

sum이라는 i32형 변수를 선언해보자. i32는 32비트 정수형 타입이다.

fn main() {
    let sum:i32=0;   
    println!("sum = {}",sum);
}

초깃값이 주어질 때는 변수 타입은 생략될 수 있다. 초기값의 형태에 따라 그 타입이 결정될 수 있기 때문이다. Rust 컴파일러가 초깃값을 보고 타입을 자동 추론하는 것이다. 아래 코드는 초기값이 0이라는 정수이기에, 타입 지정이 없어도 sum 변수는 i32 타입이 된다.

let sum=0;

초깃값이 주어지지 않는다면 변수의 타입을 생략할 수 없다. 이 경우 컴파일러가 변수의 타입을 추론할 어떠한 방법도 없기 때문에 에러가 발생한다.

let sum;  //error

변수 이름은 소문자를 사용하도록 권고되고 있다. 그대로 따르는게 좋다. 두 개 이상의 단어로 되어 있는 경우는 _를 사용해서 단어 사이를 연결시킨다. 공백은 허용되지 않는다. 한글 사용도 가능하다.

쓰기 가능한 변수의 선언
Rust에서 그냥 let을 사용하여 변수 선언을 하면, 해당 변수는 Read Only다. Rust는 변수를 Read Only로 사용하는 것을 디폴트로 하고 있고, 가변되어야할 변수만 let mut 선언을 해서 쓰라는 거다.

fn main() {
    let x = 5;
    println!("x: {}", x);  //5
    x = 6;  //error
    println!("x: {}", x);
}

let x = 5;라고 변수 x를 선언했다. 불변형 변수다. x = 6;이라고 변수의 값을 바꿀려고 시도했다. 컴파일 에러가 발생한다.

위 코드를 동작가능하게 하려면 let mut x = 5;라고 변경 가능한 변수로 선언해줘야 한다.

fn main() {
    let mut x = 5;
    println!("x: {}", x);  //5
    x = 6;  
    println!("x: {}", x);
}

let mut x = 5;라고 가변형 변수로 선언했다. x=6;이라고 x의 값을 변경하는 것이 가능하다.

타입 추론
Rust에서도 '타입 추론'을 사용한다.

let x = 5;  //x의 타입은 i32

let x:i32 = 5;라고 한 것과 동일하다.

여러 데이터 타입으로 자동 할당되는 예다.

let c = 'a';  //char
let s = "abc";  //&str
let v = vec![1,2,3]; //Vec<i32>

상수
Rust의 상수는 C나 Java의 상수와 유사하다. const를 이용해서 선언되며, 한 번 선언되면 절대 값을 바꿀 수 없다.

상수의 선언
const <상수명>: <타입> = <값>

상수의 선언은 const 키워드를 사용한다. 상수 이름은 모두 대문자를 사용한다. <타입>과 <값>을 생략할 수 없다. 타입은 변수와 동일하게 기본형(정수형, 부동소숫점형 등)과 문자열(&str)이 가능하다. 상수의 값으로 매크로를 사용할 수 없다.

const MAX:u32 = 1000;  //함수 밖에서 선언. 모든 함수에서 사용가능, 상수명은 대문자 사용

fn main(){
    const MIN:u32 = 1;
    println!("{}, {}", MAX, MIN);
    sub_fn();

    const MAX_FLOAT:f64 = 1000.123;
    const MAX_STR:&str = "Max String";
    const MAX_ARR:[i32; 4] = [1,2,3,4];
    println!("{}, {}, {:?}", MAX_FLOAT, MAX_STR, MAX_ARR);
}

fn sub_fn(){
    println!("{}", MAX);
}

상수와 변수의 차이
const로 선언되는 상수는 mutuable할 가능성 조차 없다. 변수는 mut 키워드에 의해 수정 가능할 수 있다. 상수는 선언될 때 타입과 크기가 결정되야 한다. 따라서, <타입>과 <값> 모두 있어야 한다. 반면에 변수는 <값>만 주어져도 강제 타입추론에 의해 타입이 결정될 수 있고, <타입>만 주어지고 <값>은 나중에 지정할 수 있다. 상수는 함수 밖에서 선언되어, 전체 함수에서 사용 가능하다. 그러나, 변수는 함수 밖에서 선언할 수 없다.
