# 3.3.1 스칼라 타입

스칼라 타입은 값이 하나인 데이터 타입이다. i32 u32와 같은 것들이 이러한 타입이고, 배열같이 여러 개 값을 저장하는 것은 '복합 타입'이라고 한다.

타입 유형 | 종류 | 설명
---|---|---
정수 타입 | i8, i16, i32, i64, isize
 u8, u16, u32, u64, usize | - 소수점이 없는 숫자
- i는 양수와 음수와 같이 부호가 있는 정수
- u는 부호가 없는 양수만을 가리키는 정수
- u8은 8비트 크기의 부호가 없는 정수
- usize는 32비트 아키텍처 컴퓨터이면 32비트 크기, 64비트 컴퓨터이면 64비트 크기
부동 소수점 타입 | f32, f64 | - f32는 32비트를 사용하는 부동 소수점 타입
- f64는 64비트 크기의 부동 소수점 타입
부울 타입 | bool | true 혹은 false
문자 타입 | char | - 4바이트 크기
- 유니코드 스칼라 값을 표현 (8비트 ASCII가 아님에 유의)
