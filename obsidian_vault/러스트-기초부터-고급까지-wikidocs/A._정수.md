# A. 정수

Rust의 정수 타입을 나타내는 키워드는 직관적이다. u8이면 unsigned 8 bit이고, i16이면 signed 16 bit이다. 음수까지 표현해야한다면 i32를, 그냥 양수만을 표현한다면 u32를 주로 사용한다.
정수형의 기본 데이터 타입은 i32이다. 따라서, let x = 5;와 같이 타입을 지정하지 않으면 5의 기본 타입인 i32로 변수 x의 데이터 타입이 자동 설정된다.
let a:u32 = 2;
let b:u32 = 100;
let c:u32 = 2 * 100;  //200
let d:u32 = 2 - 100; //에러 발생. 왜냐하면 u32는 음수를 나타낼 수 없기 때문
let e:i32 = 2 - 100; // -98
let x = 5; //i32 타입으로 자동 설정된다.
usize와 isize가 특이하다. 이는 동작되는 컴퓨터의 CPU와 운영체제가 몇 비트인가에 따라 결정되는 변수이다. 32비트 컴퓨터 환경이면 usize는 32비트 크기가 된다. usize가 컴퓨터 아키텍처의 비트 크기와 같다고 했기에, 이를 다른 말로 표현하면, usize는 해당 컴퓨터에서의 포인터 크기와 같다고 할 수 있다.
Rust에서는 정수 타입에 값을 지정할 때, 십진수, 이진수 등 다양한 형태로 값을 입력할 수 있다. 10진수 ~ 2진수의 값은 모두 i32 데이터 타입이다. b'A'는 u8 타입이다. _를 숫자 사이에 넣을 수 있다는 점만 특이하고, 나머지는 다른 언어와 동일하다.
fn main(){
    //1. 가독성을 위해 숫자 사이에 _ 사용 가능
    let a1 = 100_000;
    let a2 = 100000;
    println!("{} {}",a1,a2); //100000 100000
    //2. 16진수
    let b1 = 0xff;
    let b2 = 15*16 + 15; 
    println!("{} {}",b1,b2); //255 255
    //3. 8진수
    let c1 = 0o77;
    let c2 = 7*8 + 7;
    println!("{} {}",c1,c2); //63 63
    //4. 이진수
    let d1 = 0b1111_0000;
    let d2 = 128+64+32+16;
    println!("{} {}",d1,d2); //240 240
    //5. 문자 아스키 값
    let e1 = b'A'; //e1:u8
    let e2 = 'A';  //e2:char
    println!("{} {}", e1, e2); //65 A
}
마지막 5번 예제에서 b'A'는 u8 타입의 정수이고, 'A'는 char 타입으로 설정되는 것에 유의
