# 1.2 VS Code 설치하기

Rust 코드를 짜기 위해서는 다른 편집기를 사용할 수도 있으나, VS Code 사용을 권고한다.
1) 아래 사이트에서 다운로드해서 설치.
https://code.visualstudio.com/
2) VS Code에서 Rust 디버깅을 위한 확장 툴 설치
디버깅을 위해서는 ms-vscode.cpptools가 필요하다. 아래 사이트에서 "Install"을 눌러 설치한다.
https://marketplace.visualstudio.com/items?itemName=ms-vscode.cpptools
VS Code 좌측 확장프로그램 탭에서 rust-analyzer 검색하여 설치.
tip
VS Code에서의 Rust 개발 관련한 상세 내용은 아래 페이지 참조.
https://code.visualstudio.com/docs/languages/rust
