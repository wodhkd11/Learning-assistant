# 03. 초급: Rust 기본 문법

이 챕터에서는 Rust에서의 기본 문법을 다룹니다. C, Java, Python에서도 공통인, 모든 프로그래밍 언어는 가지고 있는 공통 기능들을 다룹니다. 주석, 변수선언, for 제어문 들이 이러한 항목입니다. Rust에서 이러한 공통 기능들에 대한 사용은, C의 문법과 유사합니다. 물론 다른 점도 많지만. 각 항목별로 사용법이 어떤지, 기존 언어와는 어떻게 다른지를 중점으로 해서 살펴보면 됩니다.
