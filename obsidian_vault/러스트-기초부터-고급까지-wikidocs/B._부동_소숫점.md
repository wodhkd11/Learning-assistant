# B. 부동 소숫점

Rust에서는 부동소수점 연산을 위한 데이터 타입으로 2가지를 지원한다. Rust에서 소숫점이 있는 값에 대한 디폴트는 f64다. 따라서, let x = 2.0;이라고 하면 x의 타입은 f64가 된다. 현재 대부분의 컴퓨터가 64비트 CPU이고, 이 경우 f64가 f32와 비슷한 성능을 보이면서도 더 정밀하게 수를 다룰 수 있기 때문에, Rust에서는 f64를 부동소숫점용 기본 타입으로 했다 한다. sqrt 등 소숫점 연산이 필요한 수학 함수들은 대부분 f32 혹은 f64 타입에 내장 메서드로 제공된다. 따라서, 어떤 정수형 변수에 대해서 제곱근을 구하고자 할 때는, f64로 형변환 한 후 계산해야한다. let sqrt_u = (u as f64).sqrt();

Copy    let x = 2.0; // f64
    let y: f32 = 3.0; // f32
    println!("x={}, y={}",x,y);

    let u:u32 = 40000;
    let sqrt_u = (u as f64).sqrt();
    println!("sqrt(u) = {}", sqrt_u);
