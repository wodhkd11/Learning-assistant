# B. 배열

Rust에서도 배열을 지원한다. 여러 값들의 집합체다.

배열의 선언은 let a = [1,2,3,4,5];와 같이 대괄호로 감싸진 값을 이용해서 선언할 수 있다. 처음에 선언될 때 배열의 크기가 결정되어야 하고, 한 번 그 크기가 결정되면 다시 크기를 조정할 수 없다. 배열에는 같은 데이터 타입만을 원소로 해야 한다. let b2 = [1, 2.0];과 같이 다른 데이터 타입을 담으면 에러가 발생한다.
배열 원소에의 접근은 인덱스를 이용한다. 인덱스는 0부터 시작한다. let a = [1,2,3,4,5];에서 a[0]의 값은 1이 된다.
배열 값 전체를 출력하려면 println!에서 {:?}를 이용한다.
배열 값 전체를 1로 하는 크기 5인 배열을 만들고 싶으면 let b = [1;5];와 같이 하면 된다. 대괄호 안의 규칙은 [값;크기]다.
배열의 일부 요소를 가리키려면 슬라이스를 사용하면 된다. 배열 a에 대해서 &a[0..2]는 a[0]과 a[1] 값을 리턴한다. 앞에 &가 붙은 것은 주소 형태로 참조한다는 의미다.
배열의 슬라이스 규칙을 예제를 통해 좀 더 알아보자. let a:[i32; 5] = [1,2,3,4,5];의 경우에,
a[0..2] = [1,2]
a[1..3] = [2,3]
a[..3] = [1,2,3]
a[1..] = [2,3,4,5]
a[..] = [1,2,3,4,5]

fn main(){
    //1. 배열의 선언과 사용
    let a = [1,2,3,4,5];
    println!("{}",a[0]);  //1
    println!("{:?}",a);  //[1, 2, 3, 4, 5]
    println!("{:?}", &a[0..2]); //[1,2]
    println!("{:?}", &a[3..]); //[4,5]

    //2. [값; 크기]로 여러 개의 값 지정
    let b = [1;5];  
    println!("{:?}",b);  //[1, 1, 1, 1, 1]

    //3. 배열 선언 후 나중에 값 지정
    let b1:[i32;5];
    b1 = [1;5];  
    println!("{:?}",b1);  //[1, 1, 1, 1, 1]

    //4. 에러 
    //let b2 = [1, 2.0]; //다른 데이터 타입

    // let x = [1,2,3];
    // println!("{}", x[3]); // 범위 밖 데이터 액세스

    //5. 벡터의 사용
    let mut v = vec![1,2,3];
    v.insert(3, 4);
    println!("{:?}",v); //[1,2,3,4]    
}

tip
Rust에서는 배열 대신에 그냥 벡터를 사용하면 된다. Rust의 벡터는 크기를 변경할 수 있고, 원소를 더 추가할 수도 있으며, 정렬 등 편리한 메서드들도 사용할 수 있다. 또한, 배열을 사용하는 것 대비 성능 저하도 거의 없다. 따라서, 굳이 배열을 사용할 필요 없다.
